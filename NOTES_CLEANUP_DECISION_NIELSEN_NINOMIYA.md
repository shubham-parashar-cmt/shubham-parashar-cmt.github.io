# Cleanup decision: Nielsen-Ninomiya / Floquet note

Verdict: KEEP, but only as an exploratory working note.

Do not feature this note on the homepage or research page. It can appear quietly under Notes.

Reason:
- The explicit winding calculation for D(omega,k)=mu-exp(i(omega-k)) is useful and correct as a one-cycle winding.
- The original version overclaimed the result as a direct Floquet Nielsen-Ninomiya evasion.
- The cleaned version makes the key distinction: this is a loop winding of a non-Hermitian frequency-momentum kernel, not a Chern number on T^2 and not a counterexample to the static Hermitian Nielsen-Ninomiya theorem.
