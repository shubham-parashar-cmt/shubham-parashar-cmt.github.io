Exploratory working note

Floquet Nielsen-Ninomiya on an extended torus

A cleaned, conservative note on a \(1+1\)d Floquet/spacetime-lattice toy kernel and its loop winding on \(S^1_\omega\times S^1_k\).

Status: cleaned working note. This is not a publication claim and should not be treated as a theorem-level counterexample to Nielsen-Ninomiya.

PDF TeX source

Purpose and cleanup verdict

The original note mixed three related but logically distinct ideas: Friedan's formulation of the Nielsen-Ninomiya theorem for a static Hermitian lattice Hamiltonian; a scalar Floquet/spacetime kernel; and the claim that a nonzero winding of this kernel should be read directly as a chiral index.

The robust statement is narrower: the scalar kernel \[ D(\omega,k)=\mu-e^{i(\omega-k)} \] defines a map to \(\mathbb C^\times\) for \(|\mu|\ne1\). Along a one-cycle on which \(\theta=\omega-k\) winds once, the phase of \(D\) has winding number \(+1\) for \(|\mu|<1\) and \(0\) for \(|\mu|>1\).

The cleaned note therefore treats the example as a compact calculation of a loop winding, not as a standalone violation of the static Hermitian Nielsen-Ninomiya theorem.

Static Nielsen-Ninomiya: what is being constrained?

A standard Nielsen-Ninomiya statement constrains a static, translationally invariant, local, Hermitian lattice Hamiltonian

\[ H=\int_{\mathrm{BZ}} d^d p\;\psi_a^\dagger(\mathbf p)K^a{}_b(\mathbf p)\psi^b(\mathbf p), \]

where \(K(\mathbf p)=K^\dagger(\mathbf p)\) is a smooth matrix on the spatial Brillouin torus. Near isolated Fermi points, the low-energy blocks look Dirac-like, and the theorem states that the total chirality in each charge sector must vanish.

The crucial point for this note is object-level bookkeeping: the theorem is about a static Hermitian \(K(\mathbf p)\) on the spatial Brillouin zone, not an arbitrary frequency-momentum kernel on an extended torus.

The Floquet/spacetime toy kernel

The toy object is the scalar kernel

\[ D(\omega,k)=\mu-e^{i(\omega-k)}. \]

Here \(\omega\) is a dimensionless frequency or quasienergy angle and \(k\) is crystal momentum, so the natural domain is \(S^1_\omega\times S^1_k\). This is not a Hermitian Hamiltonian kernel. It is better thought of as a frequency-momentum Dirac operator or inverse propagator.

Since the kernel depends only on \(\theta=\omega-k\), the useful invariant is a one-cycle winding, even though the kernel is written on a two-torus.

Correct winding computation

Restrict to a loop \(C\subset S^1_\omega\times S^1_k\) on which \(\theta=\omega-k\) winds once. Define

\[ f(\theta)=\mu-e^{i\theta}. \]

For \(|\mu|\ne1\), the loop avoids the origin in the complex plane. The winding number is

\[ \nu(C)=\frac{1}{2\pi i}\int_C d\log D =\frac{1}{2\pi i}\int_0^{2\pi} d\theta\,\frac{f'(\theta)}{f(\theta)}. \]

With \(z=e^{i\theta}\), this becomes

\[ \nu(C)=\frac{1}{2\pi i}\oint_{|z|=1}\frac{dz}{z-\mu} = \begin{cases} 1, & |\mu|<1,\\ 0, & |\mu|>1. \end{cases} \]

Reversing the loop orientation reverses the sign. In particular, winding \(\omega\) at fixed \(k\) and winding \(k\) at fixed \(\omega\) give opposite signs in the \(|\mu|<1\) phase.

What this does and does not show

The one-form

\[ a=\frac{1}{2\pi i}d\log D \]

is closed wherever \(D\ne0\), and its periods over one-cycles are integers. This is the clean mathematical content of the example.

It should not be oversold as a two-dimensional topological charge on \(S^1_\omega\times S^1_k\). Since \(\mathbb C^\times\simeq S^1\), one has \(\pi_1(\mathbb C^\times)=\mathbb Z\), supporting loop windings, but \(\pi_2(\mathbb C^\times)=0\), so there is no scalar two-dimensional winding number analogous to a Chern number.

The toy model lies outside the assumptions of static Hermitian Nielsen-Ninomiya. Its nontrivial integer is a loop winding of a frequency-momentum kernel on an extended torus, not the net chirality of a static Hamiltonian on the spatial Brillouin zone.

Relation to chiral propagation

The kernel also has a useful Fourier-series interpretation. For \(|\mu|<1\),

\[ \frac{1}{\mu-e^{i\theta}} =-e^{-i\theta}\sum_{n\ge0}\mu^n e^{-in\theta}. \]

For \(|\mu|>1\),

\[ \frac{1}{\mu-e^{i\theta}} =\frac{1}{\mu}\sum_{n\ge0}\mu^{-n}e^{in\theta}. \]

Thus the two regimes correspond to opposite choices of Fourier support along the discrete diagonal direction associated with \(\theta=\omega-k\). With fixed Fourier-transform conventions, this produces propagation along one directed lattice ray. The continuum statement \(G(x,t)\propto\theta(t)\delta(x-vt)\) should be treated as a heuristic limit, not as an exact result without specifying the regulator and conventions.

Hypotheses comparison

Static Nielsen-Ninomiya

  • Hermitian \(K(k)\) on spatial \(S^1_k\)
  • Local static Hamiltonian
  • Chirality from Dirac-cone orientation
  • No-go theorem constrains net static chirality on the spatial Brillouin zone

Floquet/spacetime toy object

  • Non-Hermitian inverse-propagator-like \(D(\omega,k)\)
  • Extended \(S^1_\omega\times S^1_k\) domain
  • Loop winding of \(D\) around \(\mathbb C^\times\)
  • Outside the static Hermitian hypotheses
